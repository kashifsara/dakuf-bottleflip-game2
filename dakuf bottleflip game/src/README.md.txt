# Dakuf: BottleFlip X

**Dakuf: BottleFlip X** is a real-money, skill-based game where players flip a bottle to earn rewards.

## Features
- Use INR to play and earn.
- Realistic physics for bottle flipping.
- Multiplier system like Aviator.

## Setup

Clone this repository and install dependencies:

```bash
git clone https://github.com/YOUR_USERNAME/dakuf-bottleflip-game.git
cd dakuf-bottleflip-game
npm install
npm start
